
# ET Reco





## Run Locally

Clone the project

```bash
  git clone https://github.com/ishan-agarwal/etreco.git
```

Go to the project directory

```bash
  cd etreco
```

Install dependencies

```bash
  python -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
```

Run the Script

```bash
  python recommend.py
```
